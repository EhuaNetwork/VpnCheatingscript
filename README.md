# VpnCheatingscript
web自动化脚本
批量注册VPN账户、刷邀请量 达到续期的目的，亲测可续半年之久。
该脚本可通用于其他批量注册的场景。

环境基于:
    php-WebDriver
    chromedriver75
    

第一步：composer install 安装依赖

第二步：打开chromedriver75.exe

第三步：执行temp.php
